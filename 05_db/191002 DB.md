## 02_SQL의 개념

* SQL이란?
  * 프로그램을 조작하기 위한 언어이다. 
  * 관계형 디비랑만 묶여 있다. 
  * 파일이 아니라 프로그램이다
  * Query : 요청 ("해줘"를 하기 위해서 배워야 하는 언어 )



* 종류
  1. DDL 정의 
     * 데이터를 저장하기 이전에 하는 것은 스키마 정의를 해야한다. 표를 만들어야하는데,
     * 한번만 하면 된다. 
     * 테이블 생성 까지가 1단계이다. 
     * 생각을 가장 오래해야한다. 
  2. DML 조작
     * 제일 빈번하게 일어난다. 
     * 회원가입 하면 저장하고 조회, 수정 하고 하는 부분
